## Question

The useful question is not "which model is best?" It is "which model is best for this workflow under these constraints?"

## Evaluation Dimensions

- Accuracy
- Latency
- Cost
- Tool-calling reliability
- Instruction following
- Safety behavior
- Operational observability

## Method

Use real enterprise tasks, not generic benchmarks alone. Include short tasks, long-context tasks, tool tasks, and refusal-policy tasks.

## Output

The end result should feed a model-routing policy, not a leaderboard screenshot.
